## 0.1.0 (2026-04-15)


### Features

* Initial release of `@nanopub/nanopub-elements`, a web components library for nanopublications
* `<nanopub-table>` component for rendering nanopublication query results
* `<nanopub-item>` component for single-nanopub rendering, with template support
* `data-bind-html` binding for sanitized HTML content
